## 🔧 开发者指南

### 前置要求
- Node.js >= 18
- Rust >= 1.70 (Tauri)
- TypeScript >= 5.0

### 安装依赖
```bash
npm install
```

### 开发环境
```bash
npm run dev
```

### 构建
```bash
npm run build
```

### 测试
```bash
npm test
```
